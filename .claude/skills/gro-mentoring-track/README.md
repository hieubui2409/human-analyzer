# gro:mentoring-track

> Document and analyze mentoring relationships using Kram's Developmental Networks framework.

## What it does

Gathers mentoring data from growth/mentoring-map.md, relationships/ files, materials, and identity/core.md family context. Analyzes mentoring relationships using **Kram's model** (career functions: sponsorship, coaching, exposure; psychosocial functions: friendship, acceptance, role model). Outputs mentoring network assessment, function completeness, diversity/strength scoring, and insights.

## When to use

- **Trigger phrases:** "mentoring track", "mentor analysis", "mentoring relationship", "kram"
- **Workflow position:** Complements career-path; feeds into PSY domain (GRO→PSY boundary)
- **Output user:** profile builders understanding support systems

## Flags

| Flag | Effect |
|------|--------|
| `--character <name>` | Analyze one character only |
| `--all` | Analyze all 3 characters (default) |
| `--json` | Machine-readable output |

## What it does NOT do

- **Not relationship psychology:** analyzes mentoring functions only; doesn't assess attachment or psychological dynamics (use PSY domain)
- **Not prescriptive:** does not recommend mentors or suggest new relationships
- **Boundary (Rule 15):** GRO→PSY one-way only; reads from PSY (family context) but never writes PSY assessment

## Tiếng Việt

**Mục đích:** Phân tích mối quan hệ cố vấn bằng mô hình Kram (các chức năng sự nghiệp + tâm lý xã hội). Xuất đánh giá mạng lưới cố vấn, điểm đa dạng/độ bền, và hiểu biết.

**Khi nào dùng:** Bổ sung career-path. Cung cấp cho lĩnh vực PSY (biên giới GRO→PSY).

**Không làm được:** Không đánh giá tâm lý mối quan hệ. Không gợi ý cố vấn. Chỉ đọc từ PSY.
