# orc:dream

> Periodic consolidation of character insights and storytelling patterns.

## What it does

Scans memory and instinct stores, merges overlapping memories, prunes stale ones, identifies contradictions, and promotes high-confidence patterns into agent memory. Includes instinct lifecycle management (decay, archive, clustering, promotion). Run weekly or when memory feels fragmented.

## When to use

- **Weekly maintenance** — keep memory system healthy
- **After 5+ sessions** — consolidate accumulated learnings
- **Memory feels scattered** — merge overlapping insights
- **High-confidence instincts ready** — promote to agent memory
- Trigger phrases: "dream", "consolidate", "clean up learnings", "memory maintenance"

## Flags

| Flag | Effect |
|------|--------|
| (none) | `--full` mode: scan + merge + prune + strengthen + instinct lifecycle (default) |
| `--scan` | Inventory memories, report health stats |
| `--merge` | Find overlapping memories, propose merges |
| `--prune` | Identify stale/outdated memories |
| `--full` | All phases in sequence including instincts (default) |
| `--instincts` | Run instinct lifecycle only |

## What it does NOT do

- Does NOT delete memories without confirmation—always shows before/after.
- Does NOT modify profile files—only memory directory.
- Does NOT execute instinct promotions—recommends; you decide.

## Tiếng Việt

### Nó làm gì

Quét bộ nhớ và kho bản năng, hợp nhất các bộ nhớ trùng lặp, cắt tỉa các bộ nhớ cũ, xác định mâu thuẫn, và thăng cấp các mẫu độ tin cậy cao vào bộ nhớ tác nhân. Bao gồm quản lý vòng đời bản năng (phân rã, lưu trữ, phân cụm, thăng cấp). Chạy hàng tuần hoặc khi bộ nhớ cảm thấy rải rác.

### Khi nào dùng

- **Bảo trì hàng tuần** — giữ hệ thống bộ nhớ khỏe mạnh
- **Sau 5+ phiên** — hợp nhất những kiến thức tích lũy
- **Bộ nhớ cảm thấy rải rác** — hợp nhất các kiến thức trùng lặp
- **Các bản năng độ tin cậy cao sẵn sàng** — thăng cấp lên bộ nhớ tác nhân
- Cụm từ kích hoạt: "dream", "consolidate", "clean up learnings", "memory maintenance"

### Không làm gì

- **Không xóa** bộ nhớ mà không xác nhận—luôn hiển thị trước/sau.
- **Không sửa đổi** tệp hồ sơ—chỉ thư mục bộ nhớ.
- **Không thực hiện** thăng cấp bản năng—khuyến nghị; bạn quyết định.
