# gro:learning-profile

> Map cognitive style and learning patterns using Kolb's Experiential Learning Cycle.

## What it does

Gathers learning data from growth/learning-profile.md, identity/core.md education, psychology/growth-edges.md, and materials. Classifies dominant Kolb style (Diverging, Assimilating, Converging, Accommodating) and identifies learning cycle strengths/gaps. Outputs learning patterns, study behaviors, and content-style recommendations.

## When to use

- **Trigger phrases:** "learning profile", "learning style", "kolb", "cognitive style"
- **Workflow position:** Pairs with competency-map; feeds into CRE content personalization
- **Output user:** profile builders designing learning/content approaches

## Flags

| Flag | Effect |
|------|--------|
| `--character <name>` | Profile one character only |
| `--all` | Profile all 3 characters (default) |
| `--json` | Machine-readable output |

## What it does NOT do

- **Not prescriptive:** does not design learning paths; only profiles current patterns
- **Not psychological diagnosis:** Kolb is learning preference, not psychological pathology
- **Boundary (Rule 15):** GRO only; growth-edges reference is read-only (no PSY judgment)

## Tiếng Việt

**Mục đích:** Phân loại phong cách Kolb (Diverging, Assimilating, Converging, Accommodating) và xác định các điểm mạnh/lỗ hổng của chu kỳ học tập. Xuất các mô hình học tập, hành vi học tập, và khuyến nghị về phong cách nội dung.

**Khi nào dùng:** Kết hợp với competency-map. Cung cấp cho CRE để cá nhân hóa nội dung.

**Không làm được:** Không thiết kế lộ trình học tập. Không chẩn đoán tâm lý. Chỉ đọc growth-edges.
