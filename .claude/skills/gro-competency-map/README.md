# gro:competency-map

> Inventory and assess skills using Dreyfus 7-level model and SFIA hybrid framework.

## What it does

Gathers skill data from growth/competencies.md, identity/core.md education/career, achievements.md, and materials. Assesses each skill against Dreyfus 7-level scale (Novice through Practical Wisdom). Outputs skill matrix with evidence, confidence scores, strength clusters, and gap recommendations.

## When to use

- **Trigger phrases:** "competency map", "skill assessment", "skill inventory", "dreyfus"
- **Workflow position:** Foundation skill; pairs with career-path for full GRO picture
- **Output user:** profile builders assessing skill development

## Flags

| Flag | Effect |
|------|--------|
| `--character <name>` | Assess one character only |
| `--all` | Assess all 3 characters (default) |
| `--gaps-only` | Show only Novice-Advanced Beginner skills (levels 1–2) |
| `--json` | Machine-readable output |

## What it does NOT do

- **Not training plans:** does not design learning paths (use GRO context + CRE for that)
- **Not prescriptive:** analyzes current state; does not recommend specific upskilling
- **Boundary (Rule 15):** GRO only; no psychological assessment of learning barriers (use PSY domain for that)

## Tiếng Việt

**Mục đích:** Đánh giá kỹ năng theo thang Dreyfus 7 cấp độ (Novice → Practical Wisdom). Xuất bảng kỹ năng với bằng chứng, điểm tin cậy, các cụm điểm mạnh, và khuyến nghị về lỗ hổng.

**Khi nào dùng:** Sau khi thu thập dữ liệu cơ sở. Kết hợp với career-path để có bức tranh GRO đầy đủ.

**Không làm được:** Không thiết kế lộ trình học tập. Không gợi ý hành động cụ thể. Không đánh giá tâm lý.
