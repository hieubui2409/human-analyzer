# gro:milestone-tracker

> Track career milestones and compare actual vs planned progression.

## What it does

Gathers milestone data from milestones.md, growth/career-path.md, and identity/achievements.md. Tracks achieved, planned, and missed milestones. Compares actual vs planned dates, identifies trajectory (on-track vs delayed), assesses next milestones, and evaluates progression against career stage expectations.

## When to use

- **Trigger phrases:** "milestone tracker", "track milestones", "career milestones", "milestone status"
- **Workflow position:** Read-only status tool; pairs with career-path and career-forecast
- **Output user:** profile builders tracking career coherence over time

## Flags

| Flag | Effect |
|------|--------|
| `--character <name>` | Track one character only |
| `--all` | Track all 3 characters (default) |
| `--pending-only` | Show only planned/unachieved milestones |
| `--json` | Machine-readable output |

## What it does NOT do

- **Not prescriptive:** does not set milestone targets or recommend new goals
- **Not timeline enforcement:** does not judge milestones as "late" or "early" — only flags status
- **Boundary (Rule 15):** GRO only; no psychological assessment of milestone delay (use PSY if needed)

## Tiếng Việt

**Mục đích:** Theo dõi các cột mốc sự nghiệp và so sánh tiến triển thực tế vs kế hoạch. Xác định lộ trình (đúng lịch vs trễ), xác định cột mốc tiếp theo, và đánh giá tiến triển.

**Khi nào dùng:** Kết hợp với career-path và career-forecast để theo dõi sự liên kết sự nghiệp.

**Không làm được:** Không đặt mục tiêu cột mốc. Không yêu cầu thực thi dòng thời gian. Không đánh giá tâm lý.
