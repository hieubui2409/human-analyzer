Disambiguation rules — the exact routing logic behind orc:intake.Quy tắc phân định — logic định tuyến chính xác đằng sau orc:intake.

  • Ingest new material:Thu nhận tư liệu mới: always start with mat:loader; integration gate at mat:indexer before any PSY work.luôn bắt đầu bằng mat:loader; cổng tích hợp tại mat:indexer trước mọi công việc PSY.
  • Profile work:Công việc hồ sơ: full formulation → psy:wave; quick completeness check → psy:health-check; consistency only → psy:crossref.formulation đầy đủ → psy:wave; kiểm nhanh tính đầy đủ → psy:health-check; chỉ nhất quán → psy:crossref.
  • Scoring:Chấm điểm: generic rubric → evl:score; psychometric battery → evl:standardize; casting decision → evl:fit; dyad compatibility → evl:compatibility.rubric tổng quát → evl:score; bộ đo tâm lý → evl:standardize; quyết định vai diễn → evl:fit; tương thích dyad → evl:compatibility.
  • Content creation:Sáng tạo nội dung: one post → cre:post-writer; many platforms at once → cre:multiplatform; adapt existing → cre:repurpose.một bài → cre:post-writer; nhiều nền tảng cùng lúc → cre:multiplatform; chuyển đổi sẵn có → cre:repurpose.
  • Crisis assessment:Đánh giá khủng hoảng: always psy:crisis-assess — output is never cached, never gated through EVL.luôn dùng psy:crisis-assess — đầu ra không bao giờ cache, không bao giờ gate qua EVL.
  • Fuzzy start:Bắt đầu mơ hồ: orc:intake classifies the task type; cre:exploring clarifies what content to write; cre:angle-discovery surfaces angles from existing evidence.orc:intake phân loại loại nhiệm vụ; cre:exploring làm rõ nên viết gì; cre:angle-discovery lộ ra góc độ từ bằng chứng hiện có.
